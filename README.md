# Spotify Login Clone

A Spotify-style login page with React frontend and Node.js backend.

## Setup

### Backend
```bash
cd backend
npm install
npm start
```
Server runs on http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm start
```
App runs on http://localhost:3000

## Features
- Spotify-style UI with Tailwind CSS
- JWT authentication
- Login/Register functionality
